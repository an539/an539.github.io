The Following projects were done during the learning process under Dr.Anthony Reeves, Cornell University.

Segmentation of Skin Lesions using Combined Adaptive Thresholding and Connected Component Analysis

This algorithm was developed to segment different skin lesions from varied image dataset. The purpose of this project is to build an automated mobile application that can be used as a diagnosing/screening tool to identify the type of rash/lesion by comparing the patient's image with different template image of different diseases

Automated Anatomical Landmark Detection in 3D Chest CT images

The primary aim of this project is to develop an automated algorithm which can identify the location Jugular notch in 3D CT images which are from the Lung Image Database Consortium(LIDC) image Collection. The Significance of this project is that this algorithm primarily aids in making several medical diagnosis. The accuracy of this algorithm in identifying the Sternal notch is expected to improve the performance of several other algorithms which would rely on location of sternal notch
